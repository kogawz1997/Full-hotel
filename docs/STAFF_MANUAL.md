# Maitri Staff Manual

## Daily flow
1. Open dashboard and confirm arrivals/departures.
2. Check reservations requiring deposit or document follow-up.
3. Run check-in when guest arrives.
4. Use room move or extend stay instead of editing raw records.
5. Checkout guests and confirm housekeeping task creation.

## Housekeeping mobile
Open `/dashboard/housekeeping/mobile` on a phone. Staff can start and complete assigned rooms from the mobile view.

## Problems
Use maintenance tickets for broken rooms and mark the room out-of-order when it cannot be sold.
